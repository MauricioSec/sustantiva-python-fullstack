Descripción del libro

Internet ha cambiado de forma radical la forma en la que nos comunicamos y accedemos a la información. En pocos años la Web ha evolucionado de páginas sencillas con pocas imágenes a contenidos dinámicos y complejos que provienen de bases de datos.

Todas las páginas web están construidas con la misma tecnología: el Lenguaje de Marcado de Hipertexto (HTML) y las Hojas de Estilo en Cascada (CSS). Este libro es adecuado para cualquiera que tenga interés en aprender a crear sus propias páginas web.

Contenido del libro

El contenido de este libro se ha estructurado en cuatro grandes bloques bien diferenciados que cubren todo el espectro del desarrollo web.

En la primera parte del libro se trabajan conceptos necesarios para desarrollar páginas web, como los principios de diseño, usabilidad y accesibilidad web. Se detalla cómo planificar el sitio web antes de escribir código.

En la segunda parte se explica a fondo HTML. Partiendo de las etiquetas básicas para definir el texto, se explican las etiquetas modernas de HTML5, incluyendo formularios, tablas y contenido multimedia.

En la tercera parte se aborda CSS, el lenguaje que permite definir el aspecto visual. Se explica cómo utilizar el color, las fuentes, los fondos y cómo definir la presentación de las tablas y los formularios.

Por último, en la cuarta parte se explica cómo publicar el sitio web en un servidor real, pasando por el concepto de estándares web generales que definen su funcionamiento como W3C.

El libro incluye numerosos ejemplos prácticos y ejercicios que permiten al lector aplicar los conocimientos adquiridos de forma inmediata en proyectos reales.
