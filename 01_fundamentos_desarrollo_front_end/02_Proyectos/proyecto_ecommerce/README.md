# E-commerce Frontend: Catálogo de Servicios SOC

## 📌 Descripción del Proyecto
Este proyecto es un Producto Mínimo Viable (MVP) del frontend para una plataforma de comercio electrónico especializada en servicios de ciberseguridad corporativa y operaciones SOC (Security Operations Center). 

Desarrollado como parte del Módulo 2 del Bootcamp Python Full Stack (Sustantiva), el objetivo principal es aplicar los fundamentos de la web semántica, el diseño responsivo (*mobile-first*) y la manipulación dinámica del Modelo de Objetos del Documento (DOM) mediante JavaScript puro.

## 🚀 Características Técnicas (Features)
* **Renderizado Dinámico:** Las tarjetas de servicios se generan dinámicamente inyectando código HTML al DOM a partir de un arreglo de objetos en JavaScript.
* **Enrutamiento Front-End:** Implementación de lectura de parámetros URL (`?id=X`) para renderizar vistas de detalle individuales utilizando un único archivo lógico (`app.js`).
* **Estado de Carrito Simulado:** Gestión de eventos tipo *click* que actualizan contadores numéricos en la interfaz (Navbar) sin necesidad de recargar la página.
* **Diseño Responsivo:** Uso del sistema de grillas de Bootstrap 5 para garantizar la correcta visualización en dispositivos móviles, tablets y monitores de escritorio.
* **Recursos Visuales Propios:** Las imágenes integradas fueron generadas mediante Inteligencia Artificial y optimizadas en resolución y peso para garantizar el rendimiento web.

## 🛠️ Stack Tecnológico
* **HTML5:** Semántica estructural (`<header>`, `<main>`, `<article>`, `<nav>`).
* **CSS3 & Bootstrap 5.3.3:** Maquetación, sistema de grillas (`row`, `col`) y componentes UI (`cards`, `badges`, `navbar`).
* **JavaScript (Vanilla/ES6+):** Lógica de negocio, manipulación del DOM y manejo de eventos.
