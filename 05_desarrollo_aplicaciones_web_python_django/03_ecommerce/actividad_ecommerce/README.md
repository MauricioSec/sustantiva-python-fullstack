# Proyecto Ecommerce - MVP Seguridad y Autenticación

## Pasos para ejecutar el proyecto
1. Activar el entorno virtual: `env\Scripts\activate` (Windows)
2. Instalar dependencias: `pip install django`
3. Aplicar migraciones: `python manage.py migrate`
4. Ejecutar el servidor: `python manage.py runserver`

## Rutas principales
* Inicio de sesión: `/login/`
* Registro de usuario: `/registro/`
* Vista protegida (Panel del usuario): `/dashboard/`
* Cierre de sesión: `/logout/`
