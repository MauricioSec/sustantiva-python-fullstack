Entregable: Módulo 3 - E-commerce CLI

Descripción del programa:
Aplicación de consola interactiva en Python que simula la lógica de un comercio electrónico. Permite consultar un catálogo de maquinaria y tecnología (valorizado en USD), buscar ítems mediante coincidencias parciales, agregar productos validando su existencia y calcular el total de un carrito temporal en memoria.

Cómo ejecutarlo:
1. Abrir la terminal de comandos.
2. Navegar hasta el directorio donde se encuentra el script.
3. Ejecutar el comando: python ecommerce_m3.py
