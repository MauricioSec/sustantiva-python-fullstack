Entregable: Actividad N° 5 - Explorando Estructuras de Datos

1. ¿Cuál estructura te pareció más flexible?
La lista me resultó la estructura más flexible para almacenar datos de forma general. Permite agregar, eliminar y modificar elementos libremente, además de mantener el orden exacto en el que ingresé la información. Esto facilita el uso de bucles y la segmentación (slicing) si necesito acceder a un rango específico de datos.

2. ¿Para qué tipo de problema usarías un diccionario en lugar de una lista?
Usaría un diccionario para representar entidades estructuradas donde los datos tienen un significado específico, en lugar de ser solo una secuencia. Por ejemplo, al registrar la telemetría de un dispositivo en la red o un reporte de vulnerabilidades. Una lista me obligaría a recordar que el índice 0 es la IP, el 1 es el puerto y el 2 es el estado. Con un diccionario, simplemente asocio claves claras ("ip_origen", "puerto", "estado") a sus valores correspondientes, haciendo que el código sea mucho más legible, directo y menos propenso a errores humanos.
